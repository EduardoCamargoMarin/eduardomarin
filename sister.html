SISTERS

Modelos disponíveis




Integração com websites

Possibilidade de integrar o projeto SISTERS em seu site profissional, com interface para o usuário poder interagir e receber informações relevantes sobre o seu trabalho.

Assistente virtual

Desenvolvidas para ajudar os usuários a navegar pelo seu site, além de mostrar a mais fácil integração em aplicação web, tem um desing amigável para interação, deixando uma experiência marcante.

Personalidades

Cada SISTER tem uma personalidade única para atender a sua necessidade profissional para integra-las no seu site, garantido uma experiência única para o usuário.

Mother

Seu modelo visual foi criado tempos depois que a ideia se concretizou, gerando assim um arquivo master que serviria de embasamento para a criação do projeto SISTERS.

MOTHER é o código legado que deu a possibilidade de se expandir a ideia de que qualquer aplicação ou site, tenha uma assistente virtual pronta para ajudar os usuários.

EVA

Primeira modelo de assistente criada após a MOTHER, sua funcionalidade principal erá ser uma assistente virtual verbosa com capacidade de se comunicar com o usuário.

Alem disso, EVA é capaz de interair com o usuário de uma maneira que possa quebrar a quarta parede.


FARAH

FARAH é a segunda das sister a ser desenvolvida, a sua principal qualidade está na não robotização das frases.

Em outras palavras, FARAH é capaz de se comunicar com o usuário sem parecer que estamos falando com uma máquina, mas sim com algo muito mais especial.


SAINT

SAINT é a terceira sister a ser criada com o propósito de atender os usuários que buscam precisão nas informações.

Ela fornece informações precisas e com pouca interação, afim de garantir conteúdos sob demanda ao usuário.


EVELINE

EVELINE é a mais nova do projeto SISTERS, com a voz mais delicada e também mais animada que as outras sisters, ela é capaz de interagir mais com o usuário.

O foco dela é fornecer o maximo de informação possível ao usuário.


LUNA

LUNA é a mais experiente de todas as sisters, ela vai providenciar informações importantes para o usuário e onde encontrar elas pelo site.

Alem disso LUNA é capaz explicar processos e meios de contato com a empresa do site a qual foi implantada.


LAYLA

LAYLA é a unica sister com sistema de randomização, ela é capaz de fornecer informações sobre o site ou sobre qualquer assunto pertinente ao ambiente o qual ela foi implantado.

Ela não segue um padrão, fazendo com que cada interação com LAYLA seja algo único.
